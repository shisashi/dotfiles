========
dotfiles
========

how to use
----------
::

  % cd ~
  % git clone git://github.com/shisashi/dotfiles
  % ln -s dotfiles/.zshrc
  % ....
  % mkdir ~/.vim
  % cd ~/.vim
  % ln -s ~/dotfiles/vimfiles/dict
  % ln -s ~/dotfiles/vimfiles/templates
  % mkdir ~/.emacs.d
  % cd ~/.emacs.d
  % ln -s ~/dotfiles/emacs/init.el
